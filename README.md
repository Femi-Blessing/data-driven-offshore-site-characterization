# Data-Driven Offshore Site Characterization

## Overview

This project explores how data-driven techniques can be used to characterize offshore soils using Cone Penetration Test (CPT) and shear-wave velocity data obtained from offshore wind investigations.

## Motivation

Reliable site characterization is critical for offshore infrastructure and future offshore wind developments.

As an aspiring offshore geotechnical engineer, I am interested in developing computational workflows that support smart site investigation and data-driven geotechnical decision making.

## Objectives

- Explore offshore CPT datasets
- Visualize soil profiles
- Investigate soil behavior patterns
- Apply data analytics techniques
- Explore data-driven site characterization methods

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
